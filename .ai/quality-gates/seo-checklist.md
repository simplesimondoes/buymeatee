# Quality Gate: SEO

- [ ] Unique page title (follows the `%s | BuyMeATee` template).
- [ ] Accurate meta description.
- [ ] Canonical URL present and correct.
- [ ] Correct indexability — no accidental `noindex`; robots agrees with intent.
- [ ] One clear H1.
- [ ] Logical heading hierarchy (no skipped levels for styling reasons).
- [ ] Important content is server rendered.
- [ ] Internal links to and from related pages with descriptive anchors.
- [ ] Descriptive alt text on meaningful images.
- [ ] Structured data matches visible content and validates.
- [ ] Page is included in the sitemap.
- [ ] No duplicate intent with an existing page.
- [ ] No keyword stuffing.
- [ ] Valid Open Graph (and Twitter/X card) output.
