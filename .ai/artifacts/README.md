# Artifacts

Working documents produced during development.

- `current/` — active discovery reports, audits, screenshots, release notes and temporary plans.
- `archive/` — superseded working material. Move things here rather than deleting them mid-stream; delete from here when genuinely obsolete.

**Permanent knowledge does not live here.** If something learned in an artifact is durable, move it to `CLAUDE.md`, `.ai/skills/`, `.ai/context/` or `arc42/`.

Do not fill the repository with unnecessary generated documents.
